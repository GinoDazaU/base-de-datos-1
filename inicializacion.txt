Comandos Básicos para el Proyecto (Inicialización Paso a Paso)

1. Ejecutar CrearDatos.py  
Ejecuta el script `CrearDatos.py` para realizar cualquier configuración adicional o tareas automatizadas relacionadas con la base de datos:
python CrearDatos.py

2. Conectar a PostgreSQL  
Abre la terminal y conecta a PostgreSQL como el usuario `postgres` (o el usuario correspondiente):
psql -U postgres

3. Crear la base de datos (si no existe)  
Una vez dentro de psql, crea la base de datos con el siguiente comando:
CREATE DATABASE proyecto;

4. Conectar a la base de datos recién creada  
Después de crear la base de datos, conéctate a ella:
\c proyecto

5. Crear un esquema (si no existe)  
Crea un esquema dentro de la base de datos. Por ejemplo, si quieres crear un esquema llamado `s10k`, usa el siguiente comando:
CREATE SCHEMA s10k;

6. Establecer el esquema de trabajo  
Cambia al esquema `s10k` para trabajar dentro de él:
SET search_path TO s10k;

7. Cargar las tablas, restricciones y triggers  
Ejecuta los scripts que contienen las definiciones de las tablas, restricciones y triggers. Asegúrate de tener los archivos `tablas.sql`, `restricciones.sql` y `triggers.sql` disponibles en el directorio donde trabajas:
\i 'tablas.sql';
\i 'restricciones.sql';
\i 'triggers.sql';

8. Cargar los datos desde archivos CSV  
Asegúrate de que los archivos CSV estén en el directorio correcto. Usa el siguiente comando para cargar los datos. Aquí tienes ejemplos de cómo cargar varias tablas desde archivos CSV:
\COPY Proveedores FROM 'datos/proveedores.csv' CSV HEADER;
\COPY Productos FROM 'datos/productos.csv' CSV HEADER;
\COPY Agua FROM 'datos/agua.csv' CSV HEADER;
\COPY Cervezas FROM 'datos/cervezas.csv' CSV HEADER;
\COPY Gaseosas FROM 'datos/gaseosas.csv' CSV HEADER;
\COPY Clientes FROM 'datos/clientes.csv' CSV HEADER;
\COPY Kiosko FROM 'datos/kioskos.csv' CSV HEADER;
\COPY Administra FROM 'datos/administra.csv' CSV HEADER;
\COPY Pedidos FROM 'datos/pedidos.csv' CSV HEADER;
\COPY Hace FROM 'datos/hace.csv' CSV HEADER;
\COPY DetallesPedido FROM 'datos/detalles_pedido.csv' CSV HEADER;
\COPY Pagos FROM 'datos/pagos.csv' CSV HEADER;
